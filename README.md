# Echofy

Echofy adalah aplikasi pemutar musik sederhana berbasis Java. Aplikasi ini memiliki fitur seperti menambahkan lagu, membuat playlist, sorting, pencarian lagu, hingga fitur Premium Plan.

## Fitur Aplikasi

- Menambahkan lagu ke daftar
- Membuat playlist
- Sorting lagu (Quick Sort)
- Pencarian lagu (Linear/Binary Search)
- Pemutaran lagu dengan jeda
- Fitur Premium Plan

## Cara Menjalankan

1. Clone repository ini:
2. 2. Buka project di IDE seperti IntelliJ IDEA atau VS Code
3. Jalankan Main.java dan ikuti petunjuk di console

## Teknologi yang Digunakan

- Java
- OOP (Class, Inheritance, Polymorphism)
- Input/Output Java
- Sorting & Searching Algorithm
